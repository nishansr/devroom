# DevRoom Extension Icon

This directory contains the extension icon. The PNG file should be 128x128 pixels for the marketplace.

To create the icon from SVG, you can use:

- ImageMagick: `convert -background none -size 128x128 chat-icon.svg chat-icon.png`
- Online converter: https://svgtopng.com/
- Design tool: Figma, Adobe Illustrator, Inkscape

For now, you can use the SVG as the icon or create a PNG version before publishing.
